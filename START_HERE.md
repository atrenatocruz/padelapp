## ⚡ Super Quick Setup

Got 5 minutes? Let's go!

1. **Install**
   ```bash
   npm install
   ```

2. **Create Supabase project**
   - Go to [supabase.com](https://supabase.com)
   - Create new project
   - Run SQL from `supabase/schema.sql`

3. **Add credentials**
   - Copy `env.example` to `.env`
   - Add your Supabase URL and key

4. **Start**
   ```bash
   npm run dev
   ```

5. **Sign up** in the app

6. **Make yourself admin**
   - Go to Supabase > Table Editor > profiles
   - Set `is_admin = true` for your user

**Done!** Create your first game! 🎉

## 🆘 Need Help?

### Common Issues

**"Supabase not configured"**
- Check your `.env` file
- Restart dev server

**"Failed to fetch"**
- Check Supabase URL is correct
- Check Supabase project is active

**OTP code not arriving**
- Check spam folder
- Try different email

**More help**: See [INSTALLATION.md](INSTALLATION.md) Troubleshooting section

## 📱 Using the App

### As a Player

1. **Sign up** with email
2. **View games** on home page
3. **Click "Quero jogar"** to join
4. Choose: **Alone** or **With partner**
5. Wait for game to fill (4 players)
6. **Play!** 🎾
7. **Submit result** after game
8. **View ranking** to see your stats

### As an Admin

1. Go to **Admin** tab
2. Click **Criar novo jogo**
3. Fill in details (date, location)
4. Click **Criar**
5. Members can now join!

**Full guide**: [GUIA_ADMIN_PT.md](GUIA_ADMIN_PT.md)
